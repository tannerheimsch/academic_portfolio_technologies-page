Executive Course Summary - Associate's Degree

This is a collective overview of all the courses I completed throughout my two-year IT-Data & Analytics Specialist program at CVTC.

Included are insights into the technologies I utilized, links to various class projects, and key learnings gleaned from the curriculum.


Intro to Computers & Programming

Overview: This introductory course provided a comprehensive look into Information Technology and software development fundamentals, covering operating systems, file management, problem-solving, and programming concepts.

Key Skills Learned:

  • Familiarity with Linux and Windows OS and their CLI functions.
  • Understanding of file management principles.
  • Proficiency in problem-solving techniques relevant to software development.
Content Covered:

CLI File Management

  • Basic file operations such as creating, moving, copying, and deleting files and directories.
  • Linux CLI commands to manipulate directories, clone files, and view information about the file system: $mkdir [directory name], $touch [file name], $mv [file name] [new location]

Introductory Look Into JavaScript

  • Fundamentals of JavaScript syntax, variables, and data types.
  • Basic control structures and functions in JavaScript for simple scripting tasks.

Learning Outcomes:

  • Confidence navigating and managing files using the command-line interface in Linux/Ubuntu environments.
  • Developed a foundational understanding of JavaScript programming, enabling me to write basic scripts and understand its relevance to data analysis tasks.
  • Became equipped with essential computer literacy skills and programming fundamentals, setting the stage for further exploration and learning in programming languages.

Portfolio Projects

  • Apache/Bash CLI Lab Assessment

  • File System Diagram - Case Project

Database I

Overview: Throughout the course, we examined the history and evolution of databases, gaining insights into their significance across different industries. We focused on the relational model, actively practicing tasks such as creating, populating, manipulating, and querying multi-table relational databases using the SQLite database.

Key Skills Learned:

  • Familiarity with relational database management principles.
  • Practical SQL skills for creating, populating, manipulating, and querying databases.
  • Understanding of current database usage in various industries.
Content Covered:

Relational Database Management:

  • Exploration of the relational model and its importance in database management.
  • Study of database evolution and history, providing insights into modern database systems.

Practical SQL Skills:

  • Hands-on experience in creating, populating, manipulating, and querying multi-table relational databases using SQLite.
  • Exploration of SQL commands and their practical applications.

Learning Outcomes:

  • Solid foundation in relational database management principles.
  • Proficiency in practical SQL skills, enabling effective database management and querying.
  • Enhanced understanding of database usage in real-world contexts across various industries.

Portfolio Projects:

  • Multi-Table SQL Queries 1

  • Multi-Table SQL Queries 2

Database II

Overview: In this sequel to Database I, I delved into more complex SQL approaches and learned the principles of vast database design. We covered an extensive exploration of SQL techniques, data normalization, and relational data modeling using tools such as Dia and MySQL.

Key Skills Learned:

  • I mastered advanced SQL techniques for efficient database management.
  • I gained an understanding of ODBC (Open Database Connectivity) connections.
  • I developed proficiency in database design methodologies for creating robust database solutions.
Content Covered:

Advanced SQL Application

  • I learned to execute complex SQL such as 'UPDATE'; 'SET; WHERE;', 'ADD CONSTRAINT, CALL(StoredProcedure);'.
  • I explored performance tuning techniques for optimizing database queries.
  • I gained insight into transaction management and concurrency control.

Database Design

  • I studied principles of normalization to reduce data redundancy and improve data integrity.
  • I utilized Entity-Relationship Diagrams (ERDs) for visualizing database structure.
  • I learned relational data modeling techniques for designing efficient database schemas.

Learning Outcomes:

  • I enhanced my proficiency in SQL, enabling me to efficiently manage databases and perform complex queries.
  • I obtained an understanding of ODBC connections, facilitating interaction with various database management systems.
  • I achieved mastery of database design principles, ensuring the creation of robust and efficient database solutions.
